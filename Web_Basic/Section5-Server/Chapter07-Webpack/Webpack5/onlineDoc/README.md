## 📦️ 启动方式

### 1. 下载依赖

```
npm i
```

### 2. 运行指令

- 开发环境启动项目：`npm start` 或 `npm run dev`
- 生产环境打包项目：`npm run build`

** 特别注意：启动文档所有目录不能有任何中文，否则会报错！**
